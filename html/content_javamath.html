Apache Commons Math ODE Solver

The Apache Commons Math library for Java includes a suite of functions for solving differential equations. The javamath command of VFGEN generates Java code that can be used with the Apache Commons Math library.

The VFGEN javamath Command

Java files that can be used with the Apache Commons Math ODE suite are created by the command

$ vfgen javamath vector_field_file.vf

The file that defines the system of differential equations is [name]ODE.java, where [name] is the name of the vector field given in the vector field file.

Options

demo If the option demo=yes is given, the Java file [name]Solver.java and the bash script build-and-run-[name]Solver.sh are created. The Java file contains a simple example of using an Apache Math Commons ODE solver to solve the differential equations; the solution is printed to the terminal. The bash script contains a few commands to build and run the solver. Experienced Java programmers can probably ignore this file, and instead use whatever IDE they are familiar with to build and run the demo program. Default: demo=no

Example

Here is a sample vector field file: pendulum.vf.
The files created by

$ vfgen javamath:demo=yes pendulum.vf

are
  • pendulumODE.java
  • pendulumSolver.java
  • build-and-run-pendulumSolver.sh
