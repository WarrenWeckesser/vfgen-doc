Julia

Julia is a programming language designed for scientific computing.

The VFGEN julia Command

Julia code that can be used with the DifferentialEquations.jl library are created by the command

$ vfgen julia vector_field_file.vf

The file that defines the system of differential equations is [name].jl, where [name] is the name of the vector field given in the vector field file.

Currently VFGEN generates a minimalist implementation for ordinary differential equations. It does not take advantage of many of the features of DifferentialEquations.jl. VFGEN does not yet generate code to handle delay equations in Julia, even though DifferentialEquations.jl can handle them.

Options

demo If the option demo=yes is given, the file [name]_demo.jl is created. This Julia program generates a plot of a solution to the differential equations. The default parameter values and initial conditions are used. Default: demo=no
func If the option func=yes is given, a function will be created for each user-defined function.
Default: func=no
