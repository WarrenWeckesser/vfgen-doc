Julia

Julia is a programming language designed for scientific computing.

The VFGEN julia Command

Julia code that can be used with the DifferentialEquations.jl library are created by the command

$ vfgen julia vector_field_file.vf

The file that defines the system of differential equations is [name].jl, where [name] is the name of the vector field given in the vector field file.

Currently VFGEN generates a minimalist implementation for ordinary and delay differential equations. It does not take advantage of many of the features of DifferentialEquations.jl.

Options

demo If the option demo=yes is given, the file [name]_demo.jl is created. This Julia program generates a plot of a solution to the differential equations. The default parameter values and initial conditions are used. Default: demo=no
func If the option func=yes is given, a function will be created for each user-defined function.
Default: func=no

Here is a sample vector field file: lorenz.vf.
The files created by

$ vfgen julia:demo=yes lorenz.vf

Example 2

The delayed logistic differential equation is $$x'(t) = r x(t) \left(1 - \frac{x(t - \tau)}{K}\right)$$ where \(r\), \(K\) and \(\tau\) are parameters of the system; \(\tau\) is the constant delay. To define the history of this system, we introduce another parameter \(x_0\), which we'll use as the default initial condition \(x(0)\). For \(t < 0\), we'll use \(x(t) = x_0 e^{rt}\). This amounts to assuming that up until \(t = 0\), the behavior of \(x(t)\) is simple exponential growth.

A vector field file for this system is delayed_logistic.vf. The default values for the parameters defined in that file are \(r = 1\), \(K = 1\), \(\tau = 1\) and \(x_0 = 0.0001\).

The files created by

$ vfgen julia:demo=yes delayed_logistic.vf

are
  • delayed_logistic.jl
  • delayed_logistic_demo.jl

The demo script accepts command line options for the absolute tolerance (abstol), relative tolerance (reltol) and the stop time (stoptime).

$ julia delayed_logistic_demo.jl stoptime=25
